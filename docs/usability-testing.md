# Usability Test Design for IMDb Project

## System Under Test

IMDb Project – Movie Browsing Web Application
URL: [https://imdb-project-frontend.onrender.com/](https://imdb-project-frontend.onrender.com/)

## Test Objective

Evaluate whether first-time users can easily browse movies, find specific movie information, and understand ratings and details. The test also assesses perceived ease of use, clarity of navigation, and overall user satisfaction.

---

## Part 1: Measures

### A. Performance Measures (Objective)

1. **Task Completion Rate** – Whether participants successfully complete each task without assistance.
2. **Time on Task** – Time (in seconds) taken to complete each task.
3. **Error Rate** – Number of observable errors (e.g., clicking incorrect links, using wrong filters).
4. **Navigation Efficiency** – Number of clicks required to reach the correct information.
5. **Task Abandonment** – Whether the participant gives up before completing a task.

### B. Preference Measures (Subjective)

1. **Perceived Ease of Use** – Rated on a 5-point Likert scale (1 = very difficult, 5 = very easy).
2. **Clarity of Information** – How clear and understandable movie details are (1–5 scale).
3. **Navigation Satisfaction** – Satisfaction with menus, search, and layout (1–5 scale).
4. **Visual Appeal** – Perceived attractiveness of the interface (1–5 scale).
5. **Overall Satisfaction** – Overall experience using the website (1–5 scale).

Preference measures would be collected through a short post-test questionnaire after task completion.

---

## Part 2: Usability Test Script

### A. Introduction (Read Aloud by Moderator)

“Thank you for participating. This session tests the website, not you. There are no right or wrong answers. Please complete the tasks as naturally as possible and think aloud while doing so.”

---

### B. Scenario

“You are looking for a movie to watch and want to use an online movie database to explore options, check ratings, and read details before deciding.”

---

### C. Tasks

**Task 1: Browse Movies**
Explore the homepage and identify at least one movie that interests you.

**Task 2: Find a Specific Movie**
Find the movie *Inception* using the website.

**Task 3: View Movie Details**
Open the movie page and locate the rating, release year, and short description.

**Task 4: Compare Movies**
Return to the movie list and find another movie with a higher rating than *Inception*.

**Task 5: Navigate Back to Homepage**
Return to the homepage using the site’s navigation.

---

### D. Post-Test Questions

1. What did you find easiest about using the website?
2. What, if anything, was confusing or difficult?
3. Did you trust the ratings and information presented?
4. What improvements would you suggest?

---

## Notes

This usability test is designed but not conducted. No users are recruited and no data is collected as part of this assignment.
